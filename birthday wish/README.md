# Birthday Wishes Generator

A user-friendly website for creating and sharing personalized birthday wishes with animations, photos, and videos.

## Features

- Input birthday person's name and age
- Add personal message/notes
- Upload photos and videos
- Animated birthday greetings with cake and candle effects
- Festive decorations including balloons, gift boxes, party hats, and confetti
- Purple and black color theme for elegant birthday celebrations
- Responsive design for mobile and desktop
- Copy link functionality to share wishes
- Easy navigation to create multiple wishes

## How to Use

1. Open `index.html` in any web browser
2. Fill in the birthday person's details:
   - Name
   - Age
   - Personal message (optional)
   - Photo (optional)
   - Video (optional)
3. Click "Create Birthday Wish" to generate the personalized page
4. Use the "Copy Wish Link" button to share the wish
5. Use "Create Another" to make a new birthday wish
